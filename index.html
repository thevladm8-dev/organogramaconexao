⚙️ Configuração Inicial

Para sincronização em tempo real entre todos os usuários, é necessário configurar uma chave do JSONBin.io (serviço gratuito). Faça isso uma única vez — depois todos abrirão o arquivo normalmente.

1. Crie sua conta gratuita

Acesse jsonbin.io, clique em Sign Up e crie uma conta grátis.

2. Copie sua API Key

Após entrar, clique no seu perfil → API Keys → copie a chave (começa com $2a$ ou $2b$)

3. Dê um nome para identificar este organograma
